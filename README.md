# TrailMatch

Memory card matching game built for SEG3125 Assignment 3.

## How to Play

1. Pick a difficulty (Easy, Medium, Hard)
2. Pick a theme (Nature, Animals, Space)
3. Flip cards to find matching pairs
4. Try to finish in as few moves as possible

## Running Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech

- Next.js + React
- Tailwind CSS
- Deployed on Vercel
